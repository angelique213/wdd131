Souls – Scripture Generator

Angelique Legaspi

WDD 131

Overview

Site Name

Souls

A simple, peaceful scripture generator that helps users find uplifting verses or chapters for their soul.

Purpose

The purpose of Souls is to provide uplifting scripture passages that offer peace, guidance, and spiritual strength. The site will let users generate random verses or chapters and select a book to quickly find meaningful scripture to study or share.

Audience

The intended audience includes individuals seeking daily spiritual inspiration, students who want a simple scripture reference tool, and anyone looking for peace and light in God’s word. The design will be calm, minimal, and easy to use on both mobile and desktop devices.

Dynamic Elements (JavaScript)

JavaScript will power the scripture generator by handling user actions and updating the page dynamically. Users will be able to:

  • Click a button to receive a random scripture verse or chapter.
  • Select whether to generate a chapter or a verse.
  • Select a book to generate a verse or chapter.
  • Save verses to a personal journal and write notes about them.

The project will use objects to represent verses and scripture data, arrays to store collections of verses, and array methods like map, filter, and forEach to process and display that data. ES modules will be used to organize the JavaScript into separate files (for example, one module for data and another for UI logic). Conditional statements will be used to control random selection, handle missing data, validate user choices, and show different messages depending on what the user does.

Branding

Website Logo

The logo represents Jesus Christ walking toward light and mountains, symbolizing the journey of the soul toward truth and peace.

Typography

Heading Font: Cinzel – used for the logo, main title, and section headings to give a classic, sacred feeling.

Body Font: Inter – used for all body text to keep the content clean, modern, and easy to read.

Content

Home Page

The home page will introduce the Souls scripture generator in a simple and calming layout. A hero section will display the logo, the title “Souls,” and a short tagline such as “Words that lift the soul.” Below the hero section, a main card will invite the user to generate a scripture. When the user clicks the button, a random scripture verse and its reference will appear in the card.

The home page will also include simple controls that allow users to select a book and choose whether they want a verse or a chapter. After they click the button, the result card will display the scripture, the reference, and optional actions such as copy and “Add to favorites.” The focus of the home page is to give users immediate access to inspiration with minimal distractions.

Journal Page

The Journal page will allow users to view the scripture verses or chapters they have saved and write personal notes or reflections about each one. Each saved scripture will appear in a card with the verse text, reference, and a text area or input for journaling. Users will be able to edit or update their notes so they can record impressions from their scripture study over time.

Additional Text / Labels

  • “Generate Scripture” – main button on the home page.
  • “Click the button to receive a scripture.” – default message in the verse card.
  • “Choose a book and type to receive a verse or chapter.” – instructions on the home page controls.
  • “Saved Scriptures & Journal” – heading on the Journal page.
  • Footer text: “© 2025 Souls — Guided by Scripture”
