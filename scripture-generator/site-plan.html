Souls – Scripture Generator

Site Plan

WDD 131 | Angelique Legaspi

Site Name

Souls

A simple, peaceful scripture generator that helps users find uplifting verses for their soul.

Purpose

The purpose of Souls is to provide uplifting scripture passages that offer peace, guidance, and spiritual strength. The site will let users generate random verses or select a book and chapter to quickly find meaningful scripture to study or share.

Audience

The intended audience includes individuals seeking daily spiritual inspiration, students who want a simple scripture reference tool, and anyone looking for peace and light in God’s word. The design will be calm, minimal, and easy to use on both mobile and desktop devices.

Dynamic Elements (JavaScript)

JavaScript will power the scripture generator by handling user actions and updating the page dynamically. Users will be able to:

  • Click a button to receive a random scripture verse.
  • Select a book and chapter to generate a verse from that location.
  • See the DOM update as verses, references, and messages change based on their choices.

The project will use objects to represent verses and scripture data, arrays to store collections of verses, and array methods like map, filter, and forEach to process and display that data. ES modules will be used to organize the JavaScript into separate files (for example, one module for data and another for UI logic). Conditional statements will be used to control random selection, handle missing data, validate user choices, and show different messages depending on what the user does.

Content

Home Page – Souls

The home page will introduce the Souls scripture generator in a simple and calming layout. A hero section will display the logo, the title “Souls,” and a short tagline such as “Words that lift the soul.” Below the hero section, a main card will invite the user to generate a scripture. When the user clicks the button, a random verse and its reference will appear in the card. The focus of the home page is to give users immediate access to inspiration with minimal distractions.

Generator Page – Scripture Finder

The generator page will function more like a tool. It will allow users to select a book and chapter from dropdown menus and then click a button to receive a verse from that chapter. A result card will display the verse, the reference, and optional actions such as copying the verse. This page will help users who want scripture from a specific part of the Bible or another selected volume of scripture.

Additional Text / Labels

  • “Generate Scripture” – main button on the home page.
  • “Click the button to receive a scripture.” – default message in the verse card.
  • “Choose a book and chapter to receive a verse.” – instructions on the generator page.
  • “For God so loved the world…” – example verse for layout and testing.
  • Footer text: “© 2025 Souls — Guided by Scripture”

Wireframes
