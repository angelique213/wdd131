Overview

Purpose

The purpose of Souls is to provide a peaceful scripture generator where users can receive uplifting verses or chapters. The site helps individuals find spiritual guidance, support scripture study, and save personal reflections in a simple way.

Audience

The intended audience includes individuals seeking daily spiritual inspiration, students wanting quick scripture references for study, and anyone who desires a simple way to access uplifting scripture passages. The site will be designed to be easy to use on both desktop and mobile devices.

Dynamic Elements

JavaScript will provide the main interactive features on the site:

  • Generate a random scripture verse or chapter with a button click.
  • Allow the user to choose a book and select verse or chapter mode.
  • Display the selected scripture and its reference in a card.
  • Save scriptures to a Journal page and allow notes for each verse.

The project will use objects to represent scripture entries, arrays to store collections of verses and journal entries, and array methods such as forEach, map, and filter to process and display data. Conditional branching will handle whether the user requested a verse or chapter, missing selections, and different messages based on user actions. ES modules will be used to organize the logic into separate files like data, UI, and storage.

Content

Home Page

The Home page will introduce the Souls scripture generator. A hero section will display the logo, the site name "Souls," and a tagline such as "Words that lift the soul." A main scripture generator card will include controls to select a book, choose verse or chapter, and a button to generate scripture.

The generated scripture text and its reference will appear in a results area under the button. The user will also see options such as "add to favorites" and possibly a "Copy" button.

Home Page Image:

Journal Page

The Journal page will display a list of all scriptures that the user has saved. Each saved scripture will appear in a card that shows the verse text, reference, and a space for the user to write notes or reflections.

Users will be able to review, update, or expand their notes over time. This page will help them keep track of meaningful impressions from their scripture study.
